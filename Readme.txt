2. Compile:

    Must have CUDA and QT libqt4-opengl libqt4-opengl-dev installed.
    Change the ARCH number in qt_opengl_cuda.pro from "2.0" to corresponding numbers.

    To compile, run the following commands:

    qmake
    make
    ./qt_opengl_cuda
